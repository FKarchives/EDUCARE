# EDUCARE

EDUCARE is a React-based web application designed for Type 2 Diabetes patient education and behavior support. It focuses on delivering clear, structured learning and gentle daily habit tracking through a user-friendly, non-clinical interface.

The goal of the application is to improve patient understanding and support better long-term self-management through education, reflection, and consistency.

---

## Overview

EDUCARE provides users with:

- Structured educational content on Type 2 Diabetes  
- Scenario-based learning for real-world understanding  
- Daily behavioral check-ins  
- Progress tracking and insights  
- A supportive, non-clinical user experience  

The interface is designed to be calm, accessible, and easy to navigate.

---

## Key Features

### Home Dashboard
- Entry point to learning, check-ins, and progress tracking  
- Displays daily insight messages and navigation cards  

### Onboarding Flow
- Step-by-step user profiling  
- Collects basic health-related preferences and goals  
- Single-question progressive interface  

### Learning Modules
- Short-form educational content  
- Scenario-based learning for applied understanding  
- Reinforcement through simple feedback prompts  

### Daily Check-In
- Tracks medication adherence, meals, movement, and energy levels  
- Optional notes for user reflection  
- Immediate feedback after submission  
- Stores responses in application state  

### Progress Dashboard
- Summary of user activity and consistency  
- Weekly streak tracking  
- Simple insight-based feedback on behavior trends  

### Support Directory
- Displays nearby healthcare support options  
- Simplified, accessible layout for ease of use  

---

## Design System

The application uses a custom design system focused on clarity and accessibility.

### Visual Principles
- Soft, neutral color palette (greens, blues, creams)  
- Minimal contrast for a calm interface  
- Rounded UI components  
- Subtle shadows and transitions  

### Interaction Design
- Simple, step-based flows  
- Non-alarming feedback messaging  
- Consistent button and form behavior  
- Mobile-first responsive layout

## Tech Stack

- React (Vite)  
- TypeScript  
- Tailwind CSS  
- React Router  
- Context API for state management
